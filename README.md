# toolbox

* Languages
  * Go 1.19
  * Node 
  * Python3
* Developer tools
  * task (taskfile)
  * cheat 
  * git 
  * nvm 
  * nx 
  * make 
* Secrets
  * sops 
* Cloud providers
  * aws
  * az 
* Containers 
  * docker (must mount docker daemon)
  * kubectl 
  * helm 
  * helmfile 
  * istioctl 


* IaC Tools
  * Tfenv (Terraform)

  * tflint 
  * terraform-docs 
  * infracost
* Other 

  * rclone
